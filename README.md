## 🚀 Featured Project

### [Smart FIN](https://github.com/pedro-henrique-jacome-carneiro/smart-fin)

Personal and family finance management web app built with **Python, Flask, SQLAlchemy, Flask-Migrate, PostgreSQL via Supabase, Jinja2, Bootstrap, Chart.js, pytest, and GitHub Actions**.

It includes authentication, user-specific data isolation, family permissions, income and expense management, food voucher tracking, fixed expenses, monthly closing, dashboards, reports, CSV/Excel/PDF exports, avatar validation, automated tests, and CI workflow.
